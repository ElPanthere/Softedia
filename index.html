C’est quoi Softedia ?

Un annuaire design et modulable pour trouver des alternatives gratuites à des services payants. Interface claire, recherche instantanée, et panneau d’admin pour tout gérer.

Contribuer

Ce prototype sauvegarde localement. Active le mode admin (accès réservé) pour ajouter des pages, catégories et titres. Pour la prod : brancher une API et une base (par ex. Postgres) + auth serveur.
